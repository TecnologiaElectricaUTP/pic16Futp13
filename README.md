Curso Digitales 2
=================

Este repo es completamente Software Libre,pensado 
para estudiantes de Tecnologia Electrica .

###Clonar Repo 
    git clone@github.com:c1b3rh4ck/pic16Futp13.git 
    git checkout -b dev 
    cd pic16Futp13/
