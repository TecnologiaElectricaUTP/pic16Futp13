Curso Sistemas Digitales 2 : Tecnologia Electrica 2013-1

Este repositorio fue creado para compartir algunos de los ejercicios puestos en clase,los ejercicios desarrollados aqui solo intentan ser un complemento para estudiantes nuevos,y todos aquellos cursandolo. que quieran ver otras formas de programacion.

Inicialmente el curso maneja asembler para realizar las aplicaciones,luego se realizaran en lenguajes de alto nivel como C o C# para aplicaciones mas robustas,los proyectos se subirán después de terminado el plazo,
