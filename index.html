Curso Sistemas Digitales 2 : Tecnologia Electrica 2013-1

Este repositorio fue creado para compartir algunos de los ejercicios puestos en clase,los ejercicios desarrollados aqui solo intentan ser un complemento para estudiantes nuevos,y todos aquellos cursandolo. que quieran ver otras formas de programación,los que realizamos los ejercicios somos Darwin Salaza y Hector Jimenez.

Feedback

Todos los comentarios y sugerencias sobre este material pueden ser enviadas a "hfjimenez@utp.edu.co" o "darsalazar@utp.edu.co", antes de enviar cualquier sugerencia o comentario realiza la debida investigación en una biblioteca o internet como es debido,no lo sabemos todo solo somos estudiantes pero si esta en nuestras posibilidades trataremos de ayudarte.

Inicialmente el curso maneja asm para efectos practicos de conocer la arquitectura interna del pic,luego se realizaran en lenguajes de alto nivel como C o C# para aplicaciones mas robustas,los proyectos se subirán después de terminado el plazo.

Clonar Repo

git clone@github.com:c1b3rh4ck/pic16Futp13.git 
git checkout -b dev 
cd pic16Futp13/

Compilar Debian Gnu/Linux

Tratamos de utilizar las herramientes libres por lo que usamos mucho la suite Gputils, tu puedes compilar tu codigo asm en Gnu/linux usando Gputils(gpasm,gplink,gplib) el tambien esta disponible para windows,este mismo codigo es posible compilarlo con Mpasm.
